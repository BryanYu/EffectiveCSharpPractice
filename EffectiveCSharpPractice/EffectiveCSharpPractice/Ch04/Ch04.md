# Ch04. 用內插字串取代string.Format()

內插字串
```
var name = "test";
Console.WriteLine($"Hello,{name}");
```

string.Format()
```
var name = "test";
Console.WriteLine(string.Format("Hello,{0}",name));
```

## 優點
* 可讀性更高的程式
* 讓編譯器提供更好的靜態型別檢查

## 缺點
* 在迴圈中使用時，value type會被轉換成object，對效能有影響


## 最好作法
* 使用時轉為一般字串，避免轉型引起效能問題
* 建構SQL查詢時，不會替換成參數化查詢，還是SQL字串，所以還是要特別注意。
